2025-11-01T11:52:03.377Z | Alma
think about charm and vanna this way:

CHARM
delta change in time.

If the contract is OTM, the charm will push the delta towards zero, bcs there is less and less chance for the contract to expire ITM.
Simple math: call has positive delta. To push it towards zero, you need to add negative value to it. So OTM call charm is negative.
put has negative delta. To push its delta towards zero, you need to add a positive number to it. So OTM put charm is positive.

For ITM contracts, you want to push the delta towards one. Same logic from here.

2025-11-01T12:03:32.373Z | Alma
VANNA

take vanna as the delta change to the PROBABILITY distribution that the contract will expire ITM.
Why? bcs volatility is the SD of log returns, and implied vol is the bet on future volatility.
See my educational post about volatility.
Higher vol = wider range of returns,
lower vol = narrower range of returns.

So for an OTM call, if IV goes down, the probability to expire ITM will diminish, bcs the market estimates a lower IV for that contract as rational.
So IV goes down... delta should go down to, in order to express the diminishing probability.
So vanna must be positive, right? Bcs IV down means you extract the vanna value from the delta value.

And if IV goes up, so the market starts to assign higher chance for the contract to finish in ITM, expecting wider and faster spot movements - the delta also should go up.
Mathematically: IV up, you add the vanna value to the delta value, and bcs vanna value is a positive number, the delta will increase.
Simple, right?

For an OTM put, the delta is negative, so everything turns around, like when you mirror a function multiplying by -1
Same logic here, but don't forget that in negative territory, the zero is upward!

So if IV goes down, the delta should go towards zero... meaning, it should go upward, bcs the delta is negative.
IV down, meaning you have to extract the vanna value from the delta value.
So vanna value must be a negative number, bcs extracting a negative number is an basically addition.

And if IV goes up, the delta should go away from zero, meaning it should go down, as delta is negative.
So IV up, you should add the vanna value to the delta value...
and if vanna value is negative, adding a negative number is an extraction basically, so the delta will be more negative ("shorter").

(or in otherwords: the absolute value of delta will increase)

so vanna is negative for OTM puts
